API-Docs

Unter /data/data.json befinden sich alle Arbeiten in einer Liste. Jeder Eintrag hat folgende Informationen:

  • id (int): Eindeutige ID der Arbeit
  • subject (str): Fach, in dem die Arbeit geschrieben wurde
  • teacher (str): Lehrer, der die Arbeit aufgesetzt hat
  • grade_level (int): Stufe, in der die Arbeit geschrieben wurde
  • year (int): Jahr, in dem die Arbeit geschrieben wurde
  • topics (str-liste): Liste an Themen
  • images (int-liste): Name der Bilder
  • tags (str-liste): Tags, die für eine bessere Suche benutzt werden können (Noch nicht in Verwendung)
Alle Bilder befinden sich im Ordner /data/ und sind nach dem Schema exam-{ID der Arbeit}-{Bild ID}.{Dateiendung}

Beispiel: 

{
    "id": 15,
    "subject": "Mathe",
    "teacher": "Frau Mueller",
    "grade_level": 10,
    "year": 2024,
    "topics": [
        "Ableitung",
        "Vektoren"
    ],
    "images": [
        "exam-15-1.jpg",
        "exam-15-2.jpg",
        "exam-15-3.jpg",
        "exam-15-4.jpg"
    ],
    "tags": []
}

Die Bilder zu dieser Arbeit befinden sich also an den Pfaden:
/data/exam-15-1.jpg, /data/exam-15-2.jpg, /data/exam-15-3.jpg und /data/exam-15-4.jpg.

Arbeitenarchiv ist eine inoffizielle Sammlung von Arbeiten, die von Schülern des Gym Karlsbads erstellt wurde. Diese Seite ist nicht mit der Schule verbunden.

Die Arbeiten sind urheberrechtlich geschützt und dürfen nur für den persönlichen Gebrauch verwendet werden.
